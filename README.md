The smart glove project is designed to help mute individuals communicate by converting hand gestures into speech. It can be implemented in two ways using the ESP32 and an Arduino-based setup. In the first approach, flex sensors attached to the fingers detect bending, and the ESP32 processes these signals and sends the identified gesture wirelessly to a computer over WiFi. A Python program running on the computer receives this data and converts the corresponding text into speech using the system’s speaker, enabling real-time communication. In the second approach, an Arduino is used along with an SD card module to store pre-recorded audio files. When a gesture is detected, the Arduino directly plays the corresponding audio file through a speaker using an amplifier, making the system fully standalone without the need for a computer. Both approaches demonstrate effective gesture recognition and speech generation, with the ESP32-PC method offering flexibility and the Arduino-SD card method providing portability.
